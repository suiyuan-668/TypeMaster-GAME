# TYPE MASTER GAME

![Version](https://img.shields.io/badge/Version-1.0.0-blue)
![Author](https://img.shields.io/badge/Author-suiyuan--668-orange)

**TYPE MASTER GAME** 是一款极简主义、高性能的 Web 端打字练习游戏。

---

## ✨ 项目亮点

* **多维度练习模式**：
    * **练习模式**：不限时间，支持退格修改，适合初学者寻找手感。
    * **挑战模式**：带有评分机制与时间限制，挑战你的手速极限。
    * **严判模式 (Strike Mode)**：开启后判定更加苛刻，模拟真实输入的高压环境。
* **灵活的内容生成**：
    * **自定义导入**：支持上传 `.txt` 文档，系统将自动剔除中文，保留纯净练习内容。
    * **随机生成**：可自定义语料类型（如全英文、符号干扰等）及字数。
* **极客专属功能**：
    * **盲打模式 (Mask Mode)**：支持 0% - 100% 的遮蔽程度调节，强制记忆键位。
    * **多主题切换**：内置 **明亮、深色、霓虹** 及 **自定义颜色** 方案，满足不同审美与环境需求。
* **详细的数据反馈**：
    * 实时显示 **WPM (每分钟单词数)**、**准确率**、**Combo 连击** 及 **进度**。
    * **战绩回溯**：自动记录最近 30 条练习数据，并通过等级（SSS, SS, S, A...）直观呈现进步。

## 🚀 快速开始

由于本项目是基于 Web 技术开发，你可以直接通过浏览器运行：

1.  **克隆仓库**：
    ```bash
    git clone [https://github.com/你的用户名/TYPE-MASTER-NEO.git](https://github.com/你的用户名/TYPE-MASTER-NEO.git)
    ```
2.  **运行项目**：
    * 直接双击 `index.html` 文件即可在浏览器中打开。
    * 或者使用 VS Code 的 `Live Server` 插件以获得更好的体验。

---

## 🛠️ 技术栈

* **HTML5**：构建语义化页面结构。
* **CSS3**：实现响应式布局及动态霓虹灯交互效果。
* **JavaScript (ES6+)**：处理实时录入算法、WPM 计算及本地数据持久化。

---

## 📋 使用说明

1.  **准备阶段**：在首页选择“随机生成”语料或点击“选择文件”导入你想要练习的文档。
2.  **配置模式**：根据需求调节“盲打遮蔽程度”或开启“严判模式”，并选择你要游玩的模式。
3.  **开始打字**：确认进入后即可开始游玩，打字过程中将会实时反映你的打字进度。
4.  **查看结果**：完成后系统会弹出详细结算单，并将数据自动存入历史战绩中。
